# Agri AI - Smart Crop Disease Detection

A full-stack web application designed to help farmers detect crop diseases using AI. Includes a modern glass-morphism aesthetic, a robust TensorFlow/Keras CNN integration backend using Flask, and SQLite for user authentication.

## Included Features
1. **Authentication System**: Secure signup/login using SQLite.
2. **Dashboard**: Beautiful dashboard indicating real-time weather approximations and crop tracking.
3. **AI Crop Diagnostics**: Upload a leaf image or take a photo via camera to receive instant diagnosis, confidence score, and tailored treatments and farming tips.
4. **Responsive Premium UI**: Glassmorphic elements, modern typography (Inter), animated state transitions.

## Step-by-Step Setup Instructions

**1. Prepare Python Environment**
Ensure you have Python 3.9+ installed. It is recommended to use a virtual environment:
```bash
python -m venv venv
# On Windows
venv\Scripts\activate
# On macOS / Linux
source venv/bin/activate
```

**2. Install Dependencies**
Install Flask, TensorFlow, Pillow, and other required packages:
```bash
pip install -r requirements.txt
```

**3. Initialize Database**
Run the DB script to build the `users` table for authentication:
```bash
python init_db.py
```

**4. Generate Dummy AI Model**
To have a functioning application out of the box, we use a basic script to create an `.h5` model. This model isn't trained on robust data, but processes `(224, 224, 3)` images efficiently to demonstrate the capabilities of the application.
```bash
python train_dummy_model.py
```

**5. Start the Application**
Run the Flask server:
```bash
python app.py
```
*Note: The app runs in debug mode locally. Open your browser and navigate to `http://127.0.0.1:5000`.*

## Example Images for Testing
You can use any image with dimensions larger than 224x224 pixels representing a leaf or a plant. The dummy model will resize it to the expected shapes automatically and simulate a prediction between 3 statuses: (Healthy, Rust, Powdery Mildew).
